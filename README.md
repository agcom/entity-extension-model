# Type-mapping

Type-mapping (**TM**) is a cool [logical data model](https://en.wikipedia.org/wiki/Logical_schema). Why? Because it defines **loosely coupled** data models.

## Goals

- Reusable and extensible data models.
- Easy to learn and use.
